# Clumps

A Fabric mod that clumps XP orbs together for better visual clarity and reduced server load.

## Description

Clumps xp orbs together.

## Features

- Combines multiple XP orbs into a single visual entity
- Reduces lag from excessive particle effects
- Clean and simple mod design

## Installation

1. Install Fabric Loader
2. Install Fabric API
3. Place the `clumps` mod JAR in your mods folder
4. Launch Minecraft with Fabric
